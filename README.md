# cisv-core

Core C library for CISV (SIMD CSV parser).

## Build

```bash
make -C core all
```

## Test

```bash
make -C core test
```

## Repository map

- Core library: https://github.com/Sanix-Darker/cisv-core
- CLI: https://github.com/Sanix-Darker/cisv-cli
- PHP binding: https://github.com/Sanix-Darker/cisv-php
- Node.js binding: https://github.com/Sanix-Darker/cisv-nodejs
- Python binding: https://github.com/Sanix-Darker/cisv-python

## Release policy

This repo publishes only core source/library artifacts.
Bindings are released independently in their own repos.
